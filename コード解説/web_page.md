# web_page.h

```text
ロボットペットをスマホから操作するための「Webインターフェース（GUI）」を定義しているファイル
```

## 1. 使用されている言語

このファイルの中には、**3つの主要なWebテクノロジー**が混在している

1. **HTML5**: ページの「骨組み」を作成。ボタンや見出しなどの部品を配置
2. **CSS3**: ページの「見た目」を変更。色、透明度、ぼかし効果などを指定
3. **JavaScript**: ページの「動き」を作成。ボタンを押した瞬間にメッセージを変えるといった操作を担う

## 2. 特徴的な文法・テクニック

### ① `R"rawliteral(...)rawliteral"` (C++の生文字列リテラル)

Arduino（C++）でHTMLのような長い文字列を扱う際、通常は改行ごとに `"` を閉じたり `\n` を入れたりする必要があり非常に面倒になる

- **文法**: `R"任意の区切り文字( ... )任意の区切り文字"`
- **メリット**: この形式を使うと、HTMLを**そのままの見た目でコード内に記述**可能。コードの可読性が格段に上がる

### ② `const char html_template[] PROGMEM`

- **`PROGMEM`**: 非常に重要な組み込み用キーワード。通常、文字列はRAM（作業用メモリ）に置かれるが、これは容量を圧迫する。`PROGMEM` を使うと、文字列を**Flashメモリ（プログラム領域）に直接保存**できるため、RAMを節約しながら大きなHTMLデータを保持可能になる

### ③ CSSの「グラスモーフィズム」表現

最近のトレンドである「すりガラス風」のデザインを、CSSで高度に実装

- **`backdrop-filter: blur(10px);`**: 背景をぼかす魔法のコード
- **`rgba(255, 255, 255, 0.2)`**: 色指定に「透明度（アルファ値）」を入れることで、後ろが透けて見える半透明のボタンを作成

## 3. このコードの「動き」の解説

1. **デザイン定義**: `Press Start 2P` というレトロゲーム風のフォントをGoogleフォントから読み込んでいる
2. **ボタンの配置**: `.glass-btn` クラスでボタンの装飾を定義。`transition: 0.2s` で、タップした時に少しだけ滑らかに反応するようにしている
3. **JavaScriptの役割**:

```javascript
document.getElementById('feed-btn').addEventListener('click', function() {
    document.getElementById('status-msg').innerText = 'SENT YUMMY FEED!';
});
```

- ブラウザ（スマホ）側で、「FEED ME!」ボタンがクリックされたことを検知して、即座に画面のメッセージを書き換えてる
- これにより、**通信が届く前にユーザーへ「操作が成功した」という視覚的なフィードバックを与える**ことが可能になる
