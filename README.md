# 🚀 温湿度連動型・自律ロボットペット開発プロジェクト

本リポジトリは、温湿度センサーやシリアル通信と連動する**「卓上型の自律型ロボットペット」**の設計・組み込みソースコードを管理しています。

これまで培ったC言語によるマイコン制御の知識を活かし、限られたリソースの中で「なめらかなアニメーション」と「自律的な生命感」を追求したモノづくりを行っています。

---

## 🛠️ プロジェクト作成の目的 / 背景
* **組み込みスキルの可視化**: 各種センサーの制御知識、マイコンのメモリ制限を意識した設計、C言語による低レイヤのロジック構築など、実践的な技術力を1つの成果物として証明するため。
* **表現力の追求**: 既存の無機質な置き時計ではなく、LOVOTや配膳ロボットのような「愛着の湧くインターフェース」をハードウェアとプログラムの組み合わせで実現するため。

---

## ✨ 主な機能と技術的な見どころ

### 🤖 組み込みシステム（ロボットペットの制御ロジック）
テーブルに乗るコンパクトな置き時計サイズを想定し、OLED液晶を用いた豊かな感情表現と、センサー連動による自律動作を実装しています。

1. **数式描画によるメモリ（SRAM）節約設計**
   * 大容量の画像（ビットマップ）データをマイコン内に保持するのではなく、円や直線を描画する「数式幾何学関数」のみで表情を計算。マイコンの限られたメモリ（SRAM）を極限まで節約する設計を意識しました。
2. **`delay()` に頼らない非同期まばたきアルゴリズム**
   * ループ処理で目の縦幅を連続的に変化させ、ぬるっと閉じて開く自然なアニメーションを実装。`millis()` を活用した非同期処理（マルチタスク風設計）にすることで、描画中もセンサー値の取得やボタン入力を途切れさせずに受け付けます。
3. **乱数（`rand`）を用いた自律的な視線移動**
   * 瞳のX/Y座標を変数で動的に管理。タイマーと乱数を組み合わせることで、3〜5秒ごとにランダムな方向へ黒目がなめらかに動く「生きている感」を演出しています。
4. **確率の重み付けによるステート管理（感情分岐）**
   * `enum`（列挙型）と `switch-case` 文を用いてロボットの状態を管理。単純な均等確率ではなく、「基本は通常状態だが、数％の確率で機嫌が良くなる・悪くなる」といった重み付けシャッフルアルゴリズムをC言語で実装。
5. **外部データ（天気・気温）とのシリアル通信連動**
   * PCや温湿度センサーモジュールからのシリアル通信データ（例: 雨情報 `W_RAIN` や高温検知）をトリガーに、自動で「悲しい目（タレ目）」や「暑がっている目」へと滑らかにモーフィング（変形）する連動機能を搭載。
