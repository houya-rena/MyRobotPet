# ノイズ対策

## ハードウェア（配線）のノイズ対策まとめ

- **【対策1】電源の部屋を分ける（5V と 3.3V の分離）**
  - 元々ブレッドボードで電源を共有していたとが、OLEDディスプレイを5V、オーディオモジュールの電源を **3.3Vピン** に分離
- **【対策2】GNDの「一点アース」（根元で分ける）**
  - OLEDとスピーカーのマイナス線（GND）をブレッドボード上で合流させず、**ArduinoのそれぞれのGNDピンへ直接（別々に）繋ぐ**
  - 電気がお互いの足を引っ張り合わないようにする
- **【対策3】信号線のディスタンス（物理的距離）**
  - OLEDの通信線（SDA/SCL）から出る目に見えない電波（デジタルノイズ）をスピーカーの線が拾わないよう、ワイヤー同士を物理的に離せるように工夫

## ソフトウェア（プログラム）のノイズ対策まとめ

- **【対策4】無音時の「ピンの完全シャットダウン」**
  - 音が鳴り終わった瞬間に `pinMode(DAC_PIN, INPUT);` を実行し、ピンを電気の通らない「壁（入力モード）」に変更
  - この設定により待機時の不快な「ジーーー」というホワイトノイズを完全排除
- **【対策5】タイマーリセットによる「クジ引き暴走」のストップ**
  - `loop()` の超高速回転によって1秒間に何十回も `playCry()` が呼び出され、「バリバリバリッ」と音が重なってノイズになっていたバグを修正
  - 条件に入ったら**即座にタイマーをリセット**し、ハズレでも当たりでも次の10〜25秒間は完全にブレーキをかける鉄壁の構造に変更済

### 最終的な音声の仕組み

- **音質**：某たまごっち・某ポケモンのピカチュウを彷彿とさせる、あざと可愛くてレトロな超高音ピコピコ音。
- **静寂**：鳴いていない時は、画面がどれだけ激しく動いても完全に無音。
- **生命感**：スリープ（睡眠中）の時は静かに眠り、起きている時は10〜25秒おきに25%の確率で、その時の感情のセリフを気まぐれに呟く。

- シリアルモニターから感情を送って、目がパチパチ動くのを眺めながら、忘れた頃に「ピロリロリン♪」と鳴くという動作を体験できる
